## 1. ⚙️ 個人設定與 LINE 綁定 (必看！)

### 1️⃣ 更換個人頭像 (Avatar)
想要與眾不同嗎？您可以隨時更換大頭貼！
1.  點擊右上角的「設定」圖示。
2.  點擊目前的大頭貼，選擇 **「更換頭像」**。
3.  **預設圖示**：選擇可愛的 Emoji 作為頭像 (不用上傳)。
4.  **上傳照片**：選擇電腦中的照片，系統會開啟 **裁切工具**。
    - 支援縮放、移動，自動裁切成圓形。
    - 按下「確認上傳」即可即時更新。

### 2️⃣ 綁定 LINE 通知 (LINE Notify)
綁定後，提醒事項與薪資入帳通知會直接傳到您的 LINE！
1.  進入 **「設定」** 頁面，找到 LINE 設定區塊。
2.  **掃描 QR Code**：請掃描下方的 QR Code 加入機器人好友。
    *(請將您的 Line Bot QR Code 圖片存為 `static/img/line_qr_code.png` 以顯示於此)*
3.  **產生驗證碼**：點擊網頁上的綠色按鈕。
4.  **回傳驗證碼**：記下 6 位數代碼，在 LINE 聊天室輸入並傳送。
5.  **綁定成功**：網頁會自動更新顯示「已連結 ✅」。

## 2. 🔔 智慧提醒 (Reminders)

### 如何新增提醒？
1.  點擊上方導航列的 **「🔔 提醒」**。
2.  點擊右下角的 **「+」** 按鈕。
3.  設定內容：
    - **標題**：要做什麼事 (例如：倒垃圾)。
    - **時間**：設定提醒時間。
    - **頻率**：
        - **單次**：只會在指定日期提醒一次 (提醒後自動關閉)。
        - **每週**：可複選星期 (例如：一、三、五)。
    - **通知方式**：勾選 Line 或 Email (沒勾選的圖示會變灰色)。
4.  按下 **「儲存」**。

### 自動狀態更新
- 當單次提醒發送完畢後，系統會自動將其關閉 (變灰色)。
- 網頁每 5 秒會自動更新狀態，您不需要手動重新整理。

## 3. � 記帳管理 (Accounting)

### 開始記帳
1.  點擊導航列的 **「記帳」**。
2.  **新增支出**：點擊「+」按鈕，輸入金額、類別與備註。
3.  **預算監控**：首頁上方會顯示本月預算使用了多少百分比。
    - 如果使用超過 **80%**，進度條會變紅色警告 🚨。

### 查看報表
- 系統會自動生成 **圓餅圖** (花費類別) 與 **長條圖** (每日花費趨勢)。
- 您可以隨時下載 Excel 報表備份。

---

## 4. 💼 薪資試算 (Salary Helper)

### 設定時薪
1.  進入 **「薪資」** 頁面。
2.  點擊右上角設定，輸入您的 **時薪** 與 **月目標收入**。

### 記錄排班
1.  在行事曆上點擊日期。
2.  輸入 **上班時間** 與 **下班時間**。
3.  系統會自動計算當日的時數與預計薪資。
4.  首頁的進度條會顯示您離月目標還有多遠！🎯

## ℹ️ 附錄：系統核心技術 (System Specs)
本系統基於 Python Flask 開發，並使用了以下關鍵技術 (Library List)：

| 功能模組 | 使用套件 (Library) | 用途說明 |
| :--- | :--- | :--- |
| **核心框架** | `Flask` | 網站後端核心 |
| **資料庫** | `Flask-SQLAlchemy` | 資料儲存與管理 |
| **排程任務** | `Flask-APScheduler` | 定時發送提醒通知 |
| **Line 機器人** | `line-bot-sdk` | LINE 訊息推播整合 |
| **圖片處理** | `Pillow` | 頭貼裁切與縮圖 |
| **報表匯出** | `openpyxl` | 生成 Excel 報表 |
| **環境變數** | `python-dotenv` | 安全管理金鑰設定 |
